## Compatibility
This release is compatible with:
- i.MX 8M Plus EVK with i.MX Yocto Project BSP Rev. 5.15.72-2.2.0
- Xilinx ZCU106 with Xilinx Yocto 2023.1
- Alvium MIPI CSI-2 cameras with firmware 11.1 or higher
